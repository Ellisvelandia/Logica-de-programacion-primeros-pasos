JavaScript es un lenguaje de programación de alto nivel que se utiliza para crear páginas web interactivas y dinámicas. Es un lenguaje de programación muy utilizado para añadir funcionalidad a los sitios web, crear aplicaciones web y para móviles y crear aplicaciones del lado del servidor. JavaScript fue creado por Brendan Eich en Netscape en sólo 10 días en mayo de 1995, originalmente como un lenguaje de scripting del lado del cliente para navegadores web. Rápidamente se popularizó y más tarde fue estandarizado como ECMAScript por la Asociación Europea de Fabricantes de Ordenadores (ECMA). JavaScript es un lenguaje versátil que puede utilizarse para una gran variedad de tareas, desde sencillas animaciones de páginas web hasta complejas aplicaciones web. Es un lenguaje interpretado, lo que significa que un navegador puede ejecutar directamente el código sin compilarlo antes. De este modo, resulta muy fácil escribir y probar código rápidamente. JavaScript puede utilizarse junto con otras tecnologías web, como HTML y CSS, para crear interfaces de usuario interactivas y dinámicas. También se utiliza habitualmente en el lado del servidor con tecnologías como Node.js para crear aplicaciones web y API. JavaScript ha evolucionado a lo largo de los años, añadiendo nuevas funciones y mejoras al lenguaje con cada nueva versión. En la actualidad, es uno de los lenguajes de programación más populares del mundo, con una amplia y activa comunidad de desarrolladores que trabajan en proyectos de código abierto y crean nuevas herramientas y bibliotecas.
